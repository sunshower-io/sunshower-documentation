# Optimization Summary

# About optimization

Each instance card shows whether savings on that instance have been optimized. It will show where the instance was launched, what you will save if you switch regions, and which regions will save you money.

# Optimization drop-down menu

This drop-down menu will show you your options for launching instances. You will see not only the regions but also how much you will owe or save if you choose to launch instances out of that region.
